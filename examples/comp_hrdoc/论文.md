
## 4.3 Inter-region Reading Order Relation Prediction Module

## 4.3 区域间阅读顺序关系预测模块

**(Contextual Summary based on retrieved snippets / 基于检索内容的上下文摘要)**

**English:**
Following the Detect stage, the **Order** stage establishes the reading order relationships among the identified nodes (text regions). This process corresponds to a pre-order traversal of the hierarchical document structure tree. To achieve this, we propose a novel relation prediction model that approaches relation prediction as a dependency parsing task.

**Chinese:**
在“检测（Detect）”阶段之后，**排序（Order）** 阶段建立已识别节点（文本区域）之间的阅读顺序关系。这一过程对应于分层文档结构树的先序遍历。为了实现这一目标，我们提出了一种新颖的关系预测模型，将关系预测视为依存句法分析任务。

### 4.3.1 Multi-modal Feature Extraction Module

### 4.3.1 多模态特征提取模块

**English:**
In this module, we extract multi-modal features for each detected text region to comprehensively represent its semantic and visual information. For the Detect stage, we consider OCR'd text-lines as the basic semantic units. This module likely aggregates features from visual (image embeddings), textual (OCR text embeddings), and layout (bounding box coordinates) modalities to form a unified representation for each region.

**Chinese:**
在该模块中，我们为每个检测到的文本区域提取多模态特征，以全面表示其语义和视觉信息。对于检测阶段，我们将 OCR 识别的文本行视为基本的语义单元。该模块通常聚合来自视觉（图像嵌入）、文本（OCR 文本嵌入）和布局（边界框坐标）模态的特征，从而为每个区域形成统一的表示。

### 4.3.2 Multi-modal Feature Enhancement Module

### 4.3.2 多模态特征增强模块

**English:**
We employ a multi-modal transformer encoder to model the interactions between input pairs via a global self-attention mechanism. This enhancement module aims to capture the contextual relationships between different text regions by fusing the multi-modal features extracted in the previous step, allowing the model to understand the global document context.

**Chinese:**
我们采用多模态 Transformer 编码器，通过全局自注意力机制对输入对之间的交互进行建模。该增强模块旨在通过融合上一步提取的多模态特征来捕捉不同文本区域之间的上下文关系，使模型能够理解全局文档语境。

### 4.3.3 Inter-region Reading Order Relation Prediction Head

### 4.3.3 区域间阅读顺序关系预测头

**English:**
In response to the chain structure of reading order and the tree structure of the table of contents, we design structure-aware relation prediction models specifically tailored for these structures. This head predicts the probability of a reading order relationship existing between any two given text regions, effectively determining if one region should immediately follow another.

**Chinese:**
针对阅读顺序的链式结构和目录（Table of Contents）的树状结构，我们设计了专门针对这些结构定制的结构感知关系预测模型。该预测头用于预测任意两个给定文本区域之间存在阅读顺序关系的概率，从而有效确定一个区域是否应紧随另一个区域之后。

### 4.3.4 Inter-region Reading Order Relation Classification Head

### 4.3.4 区域间阅读顺序关系分类头

**English:**
This head is responsible for classifying the specific type of relationship between text regions (e.g., Parent-Child in a hierarchy, or Next-Sibling in reading flow). By casting the tasks uniformly as relation prediction problems, we devise an effective end-to-end solution for analyzing the hierarchical relationships within the document.

**Chinese:**
该分类头负责对文本区域之间的特定关系类型进行分类（例如层级结构中的“父-子”关系，或阅读流中的“下一个兄弟”关系）。通过将这些任务统一归纳为关系预测问题，我们设计了一种有效的端到端解决方案，用于分析文档内的分层关系。
